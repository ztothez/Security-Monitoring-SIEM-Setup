# Security Monitoring SIEM Setup

A practical project focused on collecting, processing, and analyzing security-relevant logs from critical endpoints.

The setup leverages Windows telemetry and log shippers to build a foundation for detection and investigation workflows.

## Key components
- Sysmon for detailed endpoint telemetry
- Winlogbeat and Filebeat for log collection
- Centralized log analysis for security monitoring

## Focus areas
- Authentication and system activity
- Endpoint visibility
- SIEM pipeline fundamentals

## Goal
To demonstrate how endpoint logs can be transformed into actionable security signals.
